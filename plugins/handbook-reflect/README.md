# Handbook Reflect

Reflection tools for analyzing Claude Code sessions and turning them into reusable artifacts.

## Skills

- **reflect** - Analyze a session for wrong turns and generate an interactive reflection dashboard.
- **reflect-solution** - Summarize what shipped in a session as a self-contained HTML recap deck.
- **reflect-tree** - Visualize a session as an interactive quest tree showing flow, retries, and reversals.

## Installation

```bash
/plugin marketplace add nikiforovall/claude-code-rules
/plugin install handbook-reflect
```
