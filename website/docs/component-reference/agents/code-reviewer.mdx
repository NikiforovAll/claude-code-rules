# Use `@code-reviewer` agent

<span className="badge badge--handbook-code-review">handbook-code-review</span>

Review code changes in a separate context window, implementing the Fresh Eyes Pattern.

The `@code-reviewer` agent provides systematic code analysis by running in an isolated context, free from implementation assumptions. It reviews diffs directly against a base branch, providing actionable feedback grouped by severity.

## When to Use

Use the `@code-reviewer` agent when you want to get an unbiased review of your code changes.

## Key Features

- **Separate Context**: Runs in isolated context window without implementation bias
- **Read-Only**: Does NOT modify code - only provides analysis
- **Auto-Triggering**: Automatically activates when code review is requested
- **Flexible Parameters**: Supports custom base branches

## Usage Examples

**Basic review:**
```
@code-reviewer
```

**Review against specific branch:**
```
@code-reviewer review my changes against develop
```
