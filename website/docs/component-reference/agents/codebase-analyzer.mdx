# Use `@codebase-analyzer` agent

<span className="badge badge--primary">cc-handbook</span>

Understand HOW existing code works, trace implementation details, and document technical architecture.

The `@codebase-analyzer` agent excels at analyzing implementation details, tracing data flow, identifying architectural patterns, and providing comprehensive documentation with precise file:line references. Use this agent when you need to deeply understand how existing code works without making changes or suggestions.
