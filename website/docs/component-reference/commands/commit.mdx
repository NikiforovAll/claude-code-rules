# Use `/commit`

<span className="badge badge--primary">cc-handbook</span>

Create well-formatted git commits with conventional commit messages and emojis using the `/commit` command.

This command streamlines your git workflow by automatically staging files, running pre-commit checks, and generating descriptive commit messages that follow conventional commit format.
