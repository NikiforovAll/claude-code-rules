# Use `/review`

<span className="badge badge--primary">cc-handbook</span>

Systematically review code changes against a base branch with structured, emoji-categorized feedback.

## Usage

```bash
/review [base-branch=main] [--format inline|summary] [--scope core|full]
```

**Examples:**
```bash
/review                    # Review against main, detailed format
/review develop            # Review against develop branch
/review --scope core       # Focus on core issues only
```
