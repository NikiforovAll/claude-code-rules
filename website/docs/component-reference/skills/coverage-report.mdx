# Use `coverage-report` skill

<span className="badge badge--handbook-dotnet">handbook-dotnet</span>

Generate a .NET code coverage report scoped to source files changed in the current branch. Runs tests with coverage collection and produces filtered HTML reports.

## When to Use This Skill

Use the `coverage-report` skill when you want to:

- See code coverage for only the files you changed
- Generate HTML coverage reports during PR review
- Verify test coverage before merging
