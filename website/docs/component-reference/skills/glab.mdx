# Use `glab` skill

<span className="badge badge--handbook-glab">handbook-glab</span>

Expert guidance for using the GitLab CLI (glab) to manage merge requests, issues, CI/CD pipelines, and repositories from the command line.

## When to Use This Skill

Use the `glab` skill when you want to:

- Create, review, approve, or merge GitLab merge requests
- Create and manage GitLab issues
- Monitor, trigger, or troubleshoot CI/CD pipelines
- Clone, fork, or manage repositories via CLI
- Access the GitLab API for advanced operations

## Example Queries

- "Create a merge request for this branch"
- "List my assigned issues"
- "Check the pipeline status"
- "How do I approve an MR?"
- "Show me MRs waiting for my review"
