# Use `subagent-review` skill

<span className="badge badge--handbook-code-review">handbook-code-review</span>

One-shot code review using three parallel disposable subagents that check for reuse opportunities, quality issues, and efficiency problems.

Launches three concurrent review agents, aggregates findings, and fixes issues directly. Agents are disposable — they don't persist after review.

## When to Use This Skill

Use the `subagent-review` skill when you want to:

- Quick review of recent changes
- One-shot code quality check without follow-up
- Lightweight alternative to `/team-review`
