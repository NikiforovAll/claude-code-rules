# Use `nano-banana-prompting` skill

<span className="badge badge--handbook-nano-banana">handbook-nano-banana</span>

Interactive prompt crafting assistant for Nano Banana Pro (Gemini image generation). This skill guides you through a structured process to create effective prompts by clarifying intent and applying proven techniques.

## When to Use This Skill

Use the `nano-banana-prompting` skill when you want to:

- Write effective prompts for Gemini image generation
- Get guidance on prompt structure and formatting
- Optimize existing prompts for better results
- Understand which prompting technique to use for your specific need

## Key Features

- **Interactive Q&A Process**: Uses AskUserQuestion to clarify your intent before generating prompts
- **16 Prompting Techniques**: Covers narrative, structured, photography, conceptual, and transformation approaches
- **Reference Material Support**: Handles character consistency, style references, and image blending
- **Technique Selection Guide**: Automatically recommends the best approach based on your answers
