# Use `dotnet-verify` skill

<span className="badge badge--handbook-dotnet">handbook-dotnet</span>

Manage [Verify](https://github.com/VerifyTests/Verify) snapshot tests in .NET projects — update snapshots, accept new baselines, discover verify tests, and troubleshoot mismatches.

## When to Use This Skill

Use the `dotnet-verify` skill when you want to:

- Update verified snapshots after intentional code changes
- Accept or reject new snapshot baselines
- Discover and run Verify-based tests
- Troubleshoot `.verified.txt` mismatches
